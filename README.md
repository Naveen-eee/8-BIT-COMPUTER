# 8-BIT-COMPUTER
  The objective of this project is to design, simulate, verify, and implement an 8-Bit Computer Architecture using Verilog Hardware Description Language (Verilog HDL). The architecture employs a shared-bus organization and consists of essential processor modules including the Program Counter (PC), Instruction Register (IR), Memory Address Register (MAR), Data RAM, A Register, B Register, Arithmetic Logic Unit (ALU), Flag Register, Output Register, and an FSM-based Control Unit. The design is functionally verified through simulation and targeted for FPGA implementation.
